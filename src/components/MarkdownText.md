# Markdown Previewer

## Introduction

Welcome to the Markdown Previewer! This tool will automatically convert your input to Markdown syntax.

### Cool syntax stuff:

Code, `<div></div>`, between 2 backticks.

```
//Multi-line code
function greet(name) {
  return `Hello, ${name}!`;
}
```

You can also make text **bold**

Or _italic_.

Or **_both!_**

You can also ~~cross stuff out~~.
